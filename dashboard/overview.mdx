## What is the Dashboard?

The Aegis dashboard is a sleek, futuristic web interface that lets you configure every feature of Aegis without typing a single command. Just log in with Discord, select your server, and start managing.

## Accessing the Dashboard

<Steps>
  <Step title="Visit the Dashboard">
    Go to [your-dashboard-url.com](https://your-dashboard-url.com) in your browser.
  </Step>
  <Step title="Login with Discord">
    Click the **Login with Discord** button and authorize Aegis to access your account.
  </Step>
  <Step title="Select Your Server">
    Choose the server you want to manage from the server list.
  </Step>
  <Step title="Start Configuring">
    Use the sidebar to navigate between features and configure Aegis to your liking.
  </Step>
</Steps>

<Info>
  You must have the **Manage Server** permission or higher in a server to manage it from the dashboard.
</Info>

## Requirements

- Aegis must be invited to your server before you can manage it from the dashboard
- You must be logged in with the Discord account that has the **Manage Server** permission
- Your browser must have JavaScript enabled

## Privacy & Security

- Aegis uses Discord OAuth2 for authentication — we never store your password
- Only server members with **Manage Server** permissions can access server settings
- All data is encrypted in transit via HTTPS
